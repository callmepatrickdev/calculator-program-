# Calculator App

A simple calculator built with HTML, CSS, and JavaScript. It performs basic arithmetic operations and provides a clean, responsive user interface.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- Positive/Negative Toggle (+/-)
- Clear (C)
- Error Handling
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/callmepatrickdev/calculator-program-.git
```

2. Open the project folder.

3. Run `index.html` in your browser.

## Usage

- Click the calculator buttons to enter numbers and operations.
- Press `=` to calculate the result.
- Use `%` for percentage calculations.
- Use `+/-` to change the sign of a number.
- Use `C` to clear the display.
